# BeautyGirlKotlin for Android 

### kotlin项目，一些美女图片列表，使用gank等公共api，或者解析豆瓣等页面数据抽取json，另外接入了Google的admob广告sdk
<br>


## 1.0 ~ 2.0 

常规功能开发，几个美女图片加载列表， 由于最近几个接口都失效了，准备找些新接口测试

## 3.0 

准备用 kotlin Coroutine 替换掉 rxJava



## Requirements
* Android 4.3 or higher
